# Homebrew Tap for Focusrite Volume Control

Control your Focusrite Scarlett Solo volume with your Mac's media keys.

## Install

```bash
brew tap nickmorozov/focusrite-volume-control
brew install --cask focusrite-control-2-volume
```

## Update

```bash
brew upgrade --cask focusrite-control-2-volume
```
